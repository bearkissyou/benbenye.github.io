# 我们的博客

### 初始化

0. 克隆项目
```bash
git clone -b hexo-page git@github.com:benbenye/benbenye.github.io.git
```
1. 安装主题

```bash
git submodule init
git submodule update
```

2. 安装依赖

```bash
npm i
```

3. 其他
```bash
hexo help
```

### 开始写作

1. 创建草稿

2. 编辑

3. 发布

4. 提交
```bash
git commit -m 'change'

git push
```
